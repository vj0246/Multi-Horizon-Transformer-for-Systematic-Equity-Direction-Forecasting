# Frontend — Multi-Horizon Transformer Showcase

Static Next.js 14 site (App Router, TypeScript, Tailwind, Recharts) that visualizes the
**real** backtest artifacts produced by the model pipeline. No backend.

## Data source
All charts read `public/data/*.json`, generated by:

```bash
# from repo root
python -m Source.Backtest.run
```

Do **not** hand-edit those JSON files — regenerate them from a training run.

## Develop / build

```bash
npm install
npm run dev        # http://localhost:3000
npm run build      # static export -> ./out
```

`next.config.mjs` sets `output: "export"`, so `npm run build` emits a fully static
`out/` directory suitable for any static host (deployed here on Vercel).

## Structure
- `app/page.tsx` — single-page dashboard
- `app/layout.tsx`, `app/globals.css` — shell + styling
- `components/charts.tsx` — Recharts client components
- `components/ui.tsx` — Section / Panel / Stat primitives
- `lib/data.ts` — typed imports of the JSON artifacts
- `public/data/*.json` — generated backtest results
